#Linkify

An awesome Reddit clone built in PHP, Javascript and Bootstrap by WU16.

## Getting started

### Prerequisites
- [Git](https://git-scm.com/)
- Server running Apache, MySQL and PHP7

### Installation
1. Clone the repo to your local environment.
   ```bash
   $ git clone https://github.com/lessismoreiammore/linkify.git
   ```

2. Import the `linkify.sql` file into MySQL, either through the command line or through something like PHPMyAdmin.
   ```bash
   $ mysql -u root -p linkify < linkify.sql
   ```

3. Sharing is caring! Register and post your link
